요기챗(YogiChat) 사용 가이드

관리자 페이지로 이동

요기챗은 쇼핑몰 방문 고객에게 마치 실제 상담원과 대화하는 듯한 경험을 제공하는 고도화된 AI 챗봇 솔루션입니다.
아래 주요 기능들의 실제 작동 예시를 확인해 보세요.

1. 스마트 상품 추천 기능

고객의 모호한 질문("허리 아픈데 어떤 게 좋아?")이나 요구사항을 AI가 분석하여 가장 적합한 상품을 찾습니다. 고객의 이전 구매 이력과 Cafe24 쇼핑몰에 등록된 카테고리 정보 및 상세 상품 데이터를 종합적으로 바탕으로 분석해, 맞춤형 쇼핑 링크를 제안합니다.

2. 실시간 배송 및 주문 조회

"배송조회" 요청 시, 챗봇이 로그인된 고객의 최근 주문을 조회해 송장 번호와 실시간 택배사 트래킹 링크를 즉석에서 제공합니다. (CS 업무 대폭 축소)

3. AI 지식 교육 (RAG 기반 커스텀 학습)

요기보만의 특수한 정책이나 오프라인 이벤트 등을 학습시켜 AI 답변의 퀄리티를 극대화할 수 있습니다. AI를 더욱 디벨롭하기 위해서는 관리자분들께서 각 카테고리별로 일목요연하게 새로운 정보나 정확한 정보를 지속적으로 추가(교육)해 주시는 방식으로 운영하셔야 가장 완벽한 챗봇이 완성됩니다.

  • PDF 파일 자동 조각화: PDF 업로드 시, AI가 내용을 이해하기 쉽도록 약 500자 단위로 텍스트를 나누어(Chunking) 저장합니다.
  • 지식의 시각화 (이미지 RAG): 텍스트로 설명하기 어려운 내용(사이즈 표 등)은 이미지 링크와 함께 학습시키면, 해당 키워드 질문 시 즉각 이미지를 답변으로 띄워줍니다.
  • 학습 데이터 관리: PDF에서 자동 추출되거나 이미지로 등록된 모든 학습 데이터는 관리자 페이지의 '일반문의' 탭에 모여 자동 저장되므로, 원할 때 언제든 검수하여 수정/삭제할 수 있습니다.
