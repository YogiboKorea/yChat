요기챗(YogiChat) 사용 가이드

관리자 페이지로 이동

요기챗 베타 테스트에 참여해주셔서 감사합니다! 🎉
본 챗봇은 일반적인 키워드 응답 봇이 아닌, 스스로 문맥을 판단하고 데이터를 조합하는 생성형 AI(LLM)입니다.
따라서 우리가 '얼마나 정확한 지식을 주입하느냐'에 따라 똑똑해질 수도, 엉뚱한 대답을 할 수도 있습니다. 아래 가이드를 읽고 직접 챗봇을 테스트 및 학습시켜 보세요!

1. 스마트 상품 추천 기능 (자동 연동)

고객의 모호한 질문("허리 아픈데 어떤 게 좋아?", "1인 가구 소파 추천해줘")을 AI가 분석하여 가장 적합한 상품 3가지를 추천합니다.
👉 이 부분은 직접 학습시킬 필요가 없습니다. 요기보 자사몰(Cafe24)의 카테고리와 상품 정보(이름, 가격, 썸네일)를 스스로 읽어와 분석합니다.
직원 테스트 포인트:
다양한 고객 유형(임산부, 수험생, 반려동물 가정 등)을 가정하고 질문을 던졌을 때, AI가 의도에 맞는 요기보 제품(특히 맥스 등 주력 상품)을 잘 추천하는지 확인해 주세요. 이상한 상품을 추천한다면 해당 내역을 꼭 기록해 주세요.

2. AI 지식 학습 (직접 입력 / PDF / 이미지)

우리가 직접 AI를 교육시키는 핵심 기능입니다. 고객의 자주 묻는 질문(FAQ)이나 정책 변경 사항을 관리자 페이지를 통해 챗봇의 머릿속에 집어넣습니다.
AI는 여기서 주입된 정보를 0순위로 신뢰하여 답변합니다.

  • 일반 지식(텍스트) 등록 [강력 튜닝 도구]:
    가장 직관적이고 확실한 교육 방법입니다. 특정 상황에 대한 질문(Q)과 모범 답변(A)을 직접 작성합니다.
    ex) Q: "비즈 충전은 어떻게 해?" / A: "고객센터를 통한 유상 충전 서비스와 셀프 충전 비즈 구매 두 가지 방법이 있습니다..."
  • 매뉴얼 문서 통째로 학습 (PDF RAG):
    규정집, 캠페인 설명서, 세탁 가이드 등 분량이 많은 문서를 PDF 파일로 업로드하면, AI가 이를 약 500자 단위로 자동 조각화(Chunking)하여 기억하고 있다가 고객 질문 시 해당 부분을 꺼내어 설명합니다.
  • 사진으로 보여주기 (이미지 지식):
    글로 설명하기 어려운 내용(사이즈 표, 세탁 기호, 매장 약도 등)은 "검색 키워드"와 함께 이미지를 등록하세요. 고객이 해당 키워드와 유사한 질문을 하면 AI가 답변과 함께 이미지를 띄워줍니다.
  • 수정 및 관리:
    입력한 모든 지식은 관리자 페이지 하단 탭(홈페이지, 제품정보 등)에 카드 형태로 저장됩니다. 정보가 틀렸거나 내용이 바뀌면 즉시 수정/삭제 하세요. 반영까지 약 1~2초면 충분합니다.

3. 사내 베타 테스트 미션

직원 여러분, 실제 고객의 입장이 되어 챗봇을 마음껏 괴롭혀(?) 주세요! 여러분의 테스트가 요기챗을 완벽하게 만듭니다.

[미션 1] 극한 상황 테스트:
AI가 혼란스러워할 만한 질문, 꼬투리 잡는 질문, 복합적인 질문(예: "허리 아프고 좁은 방에 쓸 빈백인데 빨간색이고 20만원 이내인 거 있어?")을 던져보고 어떻게 방어하는지 지켜봐 주세요.

[미션 2] 지식 교정 및 주입 시도:
AI가 엉뚱한 답변을 하거나 정보를 모른다고 (환각 증세) 할 경우, [통합 관리자] 페이지에 접속해 직접 정답을 'Q&A'로 등록해 보세요. 그리고 다시 똑같은 문항을 물어봤을 때 AI가 교육된 대로 답변하는지 확인합니다.

[미션 3] 피드백 수집:
테스트 중 발견한 아쉬운 점, 이상한 답변 캡처, 개선 아이디어 등은 모아서 전파해 주시면 시스템에 적극 반영하겠습니다.

상단 오른쪽의 [관리자 페이지로 이동] 버튼을 누르면 AI 지식을 학습시킬 수 있는 백오피스(chat_send.html)로 이동합니다.
